# Python & Pandas - 21-Day Refresher

**For professionals who already know Python/Pandas but feel rusty after a career break.**

This 21-day track reactivates your Python fluency for data work. Days 1-19 rebuild technical speed, Day 20 creates portfolio evidence, Day 21 prepares you for interviews.

## Day-by-Day Curriculum

| Day | Theme | Focus | Key Skills Refreshed |
|-----|-------|-------|---------------------|
| **1** | Core Python basics | Variables, lists, dicts, loops, functions | Syntax recall, basic data manipulation |
| **2** | Python control flow | Conditionals, error handling, list comprehensions | Decision logic, debugging basics |
| **3** | Functions & modules | Lambda, scope, importing, packages | Reusable code, library usage |
| **4** | NumPy arrays | Creation, indexing, vectorized operations | Array thinking, basic math ops |
| **5** | NumPy advanced | Broadcasting, universal functions, masking | Efficient computation patterns |
| **6** | Pandas Series | Creation, indexing, vectorized ops | 1D data handling |
| **7** | Pandas DataFrames | Creation, selection (.loc/.iloc), inspection | 2D data structure mastery |
| **8** | Data reading/writing | CSV, Excel, JSON, SQL basics | I/O patterns |
| **9** | Data cleaning 1 | Missing values, duplicates, data types | NA handling, basic quality checks |
| **10** | Data cleaning 2 | String operations, datetime parsing | Text and time manipulation |
| **11** | Filtering & sorting | Boolean indexing, query(), sort_values() | Data subsetting |
| **12** | Feature engineering | apply(), custom functions, binning | Derived columns |
| **13** | Grouping basics | groupby(), agg(), named aggregation | Summary statistics |
| **14** | Grouping advanced | transform(), filter(), pivot_table() | Reshaping grouped data |
| **15** | Merging data | merge(), join(), concat() | Relational operations |
| **16** | Multi-table workflows | Multiple merges, key management | Realistic ETL patterns |
| **17** | Time series basics | resample(), rolling(), shift() | Temporal analysis |
| **18** | Exploratory analysis | describe(), value_counts(), correlations | Pattern discovery |
| **19** | Visualization basics | matplotlib, seaborn (hist, scatter, box) | Communicating insights |
| **20** | Portfolio building | GitHub repo structure, README, project ideas | Showcasing your refresh |
| **21** | Interview prep | Common questions, whiteboard patterns, storytelling | Live coding confidence |

## Each Day Contains

**lesson.ipynb** - 5-10 min concept review + 2 worked examples  
**exercises.ipynb** - 3-5 practice problems (easy → medium)  
**solutions.ipynb** - Complete answers with explanations  
**reflection.md** - Template: "What felt rusty? What clicked? LinkedIn post idea?"

## Datasets Used

- **Days 1-10**: Synthetic data generated in‑notebook (customers, transactions)  
- **Days 11-19**: Semi‑realistic public datasets (Kaggle samples: retail, banking)  
- **Days 20-21**: Portfolio‑worthy project datasets (fraud detection, customer analytics)

## Prerequisites

- Python 3.8+
- Jupyter Notebook/Lab
- `pip install pandas numpy matplotlib seaborn openpyxl`

## How to Start

1. Clone this repo
2. Open `day-01-core-basics/lesson.ipynb`
3. Run cells, complete exercises, write reflection
4. Commit your work daily for portfolio evidence

**Day 1 takes 30-45 minutes. Build momentum.**
